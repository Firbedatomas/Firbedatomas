# Hola, soy Tomás Firbeda 

Soy desarrollador de software y fundador de **Frambuesa**, una plataforma para restaurantes, cafeterías y negocios gastronómicos.

Me enfoco en construir productos reales, simples y útiles, combinando **desarrollo full-stack**, **diseño de producto**, **experiencia de usuario mobile-first** e **IA aplicada a operaciones**.

## Actualmente estoy trabajando en
- **Frambuesa**: sistema para operación gastronómica, menú QR, catálogo online, ventas y herramientas para negocios.
- Mejoras en **UX/UI**, flujos operativos y automatización.
- Integraciones orientadas a simplificar la gestión diaria de comercios.
