## Building Frambuesa

**Frambuesa** is operational software for restaurants, cafés and food businesses across **LATAM** — built to run the counter, the kitchen, delivery and the back office from a single place.

It isn't a side project or a demo. It's the system I use to run my own stores, shaped by the daily reality of food service: rush hours, multiple branches, printers that have to work, and staff who need tools that are fast and obvious.

> The goal is simple: **software that survives a Saturday night.**
